# Консольный магазин на Python

Учебный проект: консольное приложение магазина с корзиной, заказами и обработкой исключений.

## Возможности

- Просмотр ассортимента товаров.
- Добавление/удаление товаров из корзины.
- Проверка остатков на складе.
- Валидация телефона (только цифры, минимум 10 символов).
- Оформление заказа.
- Сохранение покупателей в JSON и повторный вход по имени.
