Rонсольный интернет‑магазин c сохранением клиентов в JSON.

## Структура проекта

Код разделён на логические модули:

- exceptions.py — все пользовательские исключения (ShopException, InvalidDataException, и т.д.).
- models.py — основные модели домена:
  - Tovar, ElementKorziny, Korzina,
  - Pokupatel, ElementZakaza, Zakaz.
- store.py — бизнес‑логика магазина:
  - класс Magazin,
  - функции zapolnit_magazin, pokazat_tovary, restore_cart.
- clients_io.py — работа с данными клиентов:
  - load_clients(), save_clients(), файл clients.json.
- main.py — консольный интерфейс и запуск приложения (main()).
- Unittest (опционально) — юнит‑тесты для моделей и логики магазина.


## Возможности

- Просмотр ассортимента товаров (цены в тенге)
- Добавление и удаление товаров из корзины
- Проверка остатков на складе (нельзя добавить больше, чем есть)
- Оформление заказа с подсчётом итоговой суммы
- Валидация номера телефона (только цифры, минимальная длина)
- Сохранение покупателей в `clients.json` и повторный вход по имени

## Стек

- Язык: Python 3
- Хранение клиентов: JSON‑файл

## Структура классов

| Класс            | Роль                                                          |
|------------------|---------------------------------------------------------------|
| Tovar            | Товар: id, название, цена, остаток, валидация данных          |
| ElementKorziny   | Позиция в корзине: товар + количество, сумма по позиции       |
| Korzina          | Корзина покупателя: добавление, удаление, общая сумма         |
| Pokupatel        | Покупатель: id, имя, телефон, корзина, проверка телефона      |
| ElementZakaza    | Позиция в заказе: товар, количество, зафиксированная цена     |
| Zakaz            | Оформленный заказ: список позиций, статус, итоговая сумма     |
| Magazin          | Магазин: склад товаров, оформление заказов, проверка остатков |

## Пользовательские исключения

| Исключение                       | Когда возникает                                      |
|----------------------------------|------------------------------------------------------|
| InvalidDataException             | Неверные данные (цена, количество, телефон и т.д.)   |
| InsufficientQuantityException    | Недостаточно товара на складе                        |
| ItemNotFoundException            | Товар/позиция не найдены в корзине/магазине          |
| EmptyCartException               | Попытка оформить пустую корзину                      |
| ShopException                    | Базовый класс для всех ошибок магазина               |
